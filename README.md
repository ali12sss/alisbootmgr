# Ali's EFI Boot Manager

A cross-platform GUI tool to manage UEFI boot entries. Rename, reorder, delete, or change the file path of your boot entries with ease.

## 🐧 Linux (Arch/Cinnamon)
To install and add the app to your application menu:
1. `git clone https://github.com/ali12sss/alisbootmgr.git`
2. `cd alisbootmgr`
3. `chmod +x install.sh && ./install.sh`

## 🪟 Windows
You can download the ready-to-use `.exe` from the [Releases](https://github.com/YOUR_USERNAME/alisbootmgr/releases) page.
- **Note:** You must right-click and **Run as Administrator** on Windows.

## 🛠️ Building from source
Requires [Rust](https://rustup.rs/).
```bash
cargo build --release
```
